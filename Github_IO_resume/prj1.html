Identifying the Type of Breast Cancer Tumor

  • I used 3 types of Supervised Machine Learning Classification algorithms, KNN Neighbors, Logistic Regression and Linear SVC, on the "Breast Cancer Wisconsin" dataset, for identifying the breast cancer tumors either as "Malignant" or "Benign 3-32" based on the features of the dataset and compare their accuracies, after proper data pre-processing.
